  ```c++
  int num, rev=0;
  while(num>0){
      int r = num%10;
      rev = rev*10 + r;
  }
  ```

  - [ ] num is declared outside `while` body.
  - [x] Loop variable isn't updated.
  - [ ] `int r` is declared inside `while` body.
